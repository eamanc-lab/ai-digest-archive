### 6. Tokenomics: Quantifying Where Tokens Are Used in Agentic Software Engineering

**Hacker News** | Y Combinator

arXiv 论文,对 Claude Code / Cursor / Codex 等 agentic SWE 系统的 token 消耗做了量化拆解,分析了 prompt、tool-call、思考、纠错各部分各占多少 token。结论指向「工具调用反馈循环」是 token 大头,而不是用户问题本身——这对设计 agentic 系统的 context window 策略、缓存策略有直接指导意义。Top 10 中唯一的 arXiv 论文,选它是因为它直接关系到核心工作流的成本结构。

[阅读原文](https://arxiv.org/abs/2601.14470)

### 7. Sem: New primitive for code understanding – not LSPs, but entities on top of Git

**Hacker News** | Y Combinator

Ataraxy Labs 提出 `Sem`,一种新的代码理解原语:不基于 LSP(语法/编译期),而是把 Git 历史里的「实体」(函数、类、模块的演化)作为一等公民,为 AI Agent 提供更稳定的代码语义视图。LSP 在大型 monorepo 和多语言项目里启动慢、协议复杂,Sem 这条「Git-native」路线对编程 Agent 的上下文检索有潜力。值得做编程工具的人重点关注。

[阅读原文](https://ataraxy-labs.github.io/sem/)
